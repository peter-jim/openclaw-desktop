# What is Openclaw

**Openclaw** (全称：Openclaw 桌面龙虾) is a macOS/Windows desktop application that integrates AI-powered productivity features with an emotional desktop companion — a small lobster character named **Zhuazhua (爪爪)** who lives in the corner of your screen.

Built by a solo indie developer over three months, Openclaw is an experiment in the question: *can a productivity tool also have warmth?*

---

## Features

- **File Management Automation** — Smart rules to auto-organize files by type, date, or project
- **Task Automation** — Build simple workflows for repetitive desktop operations
- **AI Chat** — Integrated large language model for daily Q&A, writing, and code assistance
- **Lobster Companion (爪爪)** — An animated desktop character with idle animations: head tilts, eye blinks, tail wags. No utility. Just warmth.

---

## Who is it for

- Programmers working late nights who want a little company
- Designers and creatives with chaotic file systems
- Indie workers who want tools with personality, not just function
- Anyone who's ever felt a little lonely staring at a screen

---

## Philosophy

&gt; **Efficiency is for work. Warmth is for yourself.**  
&gt; 效率是给工作的，温度是给自己的。

Most AI tools compete on speed, capability, and model benchmarks. Openclaw competes on something harder to measure: *the feeling of not being alone while you work.*
